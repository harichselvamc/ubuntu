Image Processing API

Endpoint: /headbody

Method: POST
Description: Uploads a head and body image and combines them to create a processed image.
Parameters:
- head (file): The head image file
- body (file): The body image file
Response:
- If the request is made from a browser, it will display the processed image.
- If the request is made from Postman or a non-browser client, it will return the download link of the processed image in JSON format.

Endpoint: /removebackground

Method: POST
Description: Removes the background from an image.
Parameters:
- file (file): The image file to remove the background from
Response:
- Returns the download link of the background-removed image in JSON format.

Endpoint: /resize

Method: POST
Description: Resizes an image to a specified size.
Parameters:
- file (file): The image file to resize
Response:
- Returns the download link of the resized image in JSON format.

Endpoint: /cartoon

Method: POST
Description: Converts an image to a cartoon-like style.
Parameters:
- file (file): The image file to convert to a cartoon
Response:
- Returns the download link of the cartoonized image in JSON format.

Testing with Postman

To test the API endpoints using Postman, follow these steps:

  1. Open Postman and create a new request.
  2. Set the request method to POST.
  3. Choose the desired endpoint: /headbody, /removebackground, /resize, or /cartoon.
  4. Add the required parameters as form-data:
    • For /headbody, add head and body parameters of type File.
    • For /removebackground, /resize, and /cartoon, add a file parameter of type File.
  5. Attach the respective image files to the parameters.
  6. Send the request and view the response, which will contain the download link of the processed image.
